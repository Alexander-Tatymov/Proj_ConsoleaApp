# Как запустить TaskTracker
## Запуск из Visual Studio
1) Открыть решение TaskTracker.sln
2) Нажать F5
## Запуск из publish-сборки
1) Распаковать архив TaskTracker_v1.0.0_Фамилия.zip
2) Запустить TaskTracker.App.exe
## Где хранятся файлы
- data/tasks.json — задачи в режиме JSON
- backups/ — резервные копии
- exports/ — экспортированные файлы
- reports/ — отчёты
- logs/ — логи приложения
