# Отчёт ЛР10 — TaskTracker
## Что сделано
- Добавлен AppLogger (лог в папку logs, файл на день)
- Логируются действия:
- ADD, UPDATE, STATUS, DELETE
- BACKUP, EXPORT, IMPORT
- Добавлен пункт меню 14) показать последние строки лога
## Где лежит лог
Папка logs рядом с программой.
Файл вида: app_YYYY-MM-DD.log

## Как проверить
1) Запустить (F5)
2) Добавить задачу, изменить статус, удалить
3) Открыть logs/app_*.log и убедиться, что строки появились
4) В программе нажать 14 и увидеть последние строки
