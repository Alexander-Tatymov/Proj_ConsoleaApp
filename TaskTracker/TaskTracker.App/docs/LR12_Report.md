# Отчёт ЛР12 — интерфейсы и хранилища
## Цель
Добавить интерфейс хранения задач и возможность подмены хране
ния (JSON/Memory).
## Что сделано
- Добавлен интерфейс ITaskStorage (Load/Save)
- Реализован MemoryTaskStorage (данные в памяти)
- Реализован JsonTaskStorageAdapter (адаптер для JSON)

- Приложение запускается в двух режимах:
- JSON режим (данные сохраняются)
- Memory режим (данные пропадают после выхода)
## Как проверить
1) Запустить приложение и выбрать JSON режим
2) Добавить задачу, выйти, запустить снова -> задача осталась
3) Запустить приложение и выбрать Memory режим
4) Добавить задачу, выйти, запустить снова -> список пуст
